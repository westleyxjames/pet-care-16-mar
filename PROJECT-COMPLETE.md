## 🎨 Customization Quick Guide

### Change Colors
Find and replace in all HTML files:
```
bg-blue-600 → bg-[your-color]
text-blue-600 → text-[your-color]
from-blue-50 → from-[your-color]
```

### Change Fonts
Update in `css/styles.css`:
```css
body {
  font-family: 'Your Font', sans-serif;
}
```

### Update Images
All images use Unsplash CDN. To change:
1. Find image tags in HTML
2. Replace `src="https://images.unsplash.com/..."`
3. With your image URLs

### Modify Services
Edit `services.html`:
- Change service names
- Update descriptions
- Modify pricing
- Add/remove services
